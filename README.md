# Telecom Billing Analytics Dashboard

This project analyzes telecom billing data using Power BI.

## Dashboard Pages

### Executive Overview
Shows total revenue, ARPU, customer count, and revenue trends.

### Billing Failure Analysis
Analyzes failed payments, failure rate, and failure reasons.

### Customer Insights
Explores plan popularity, revenue by plan, and customer distribution by state.

## Tools Used

- Power BI
- DAX
- Data Visualization
- Data Modeling

## Key Insights

- Identified major payment failure reasons
- Analyzed telecom revenue trends
- Evaluated plan performance and customer distribution
